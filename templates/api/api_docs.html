API Docs

Base path: /api/
The API is not currently versioned in the URL. If you want explicit versioning later, we can add it back.

Genres

Method URL Purpose Auth Required
GET /api/genres/ Discover available genres used to categorize games. No

Games

Method URL Purpose Auth Required
GET /api/games/ Browse games. Supports filtering by ?title= and ?genre=. Includes review_avg. No
POST /api/games/ Create a new game entry (moderation-only). Yes
GET /api/games/<id>/ Fetch full details for a single game. No
PATCH / DELETE /api/games/<id>/ Update or remove a game entry (moderation-only). Yes

Events

Method URL Purpose Auth Required
GET /api/events/ Browse upcoming events. Supports filtering by ?location=. No
POST /api/events/ Create a new event (moderation-only). Yes
GET /api/events/<id>/ Fetch full details for a single event. No
PATCH / DELETE /api/events/<id>/ Update or remove an event (moderation-only). Yes

Reviews

Method URL Purpose Auth Required
GET /api/reviews/ Browse reviews (filter by game using ?game=). No
POST /api/reviews/ Create a review for a game (requires login; one review per game per user). Yes
GET /api/reviews/<id>/ Fetch full details for a single review. No
PATCH / DELETE /api/reviews/<id>/ Update or delete a review (only the author, a Moderator, or superuser). Yes

Collections

Method URL Purpose Auth Required
GET /api/collections/ List the logged-in user's game collection entries. Yes
POST /api/collections/ Add a game to the logged-in user's collection. Yes
GET /api/collections/<id>/ Retrieve a single collection entry (only your own). Yes
PATCH / DELETE /api/collections/<id>/ Update or remove a collection entry (only the owner, a Moderator, or superuser). Yes

User

Method URL Purpose Auth Required
GET /api/users/me/ Return the current user including their nested profile. Yes
PATCH /api/users/me/ Update editable fields for the current user. Yes
